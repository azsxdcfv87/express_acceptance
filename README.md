# express_acceptance

#### 安裝

1. 將此專案 Clone 到本機

2. 透過 npm 來安裝 Express

- 建立 package.json 檔

```
npm init -y
```

- 安裝所需套件

```
npm i express@4.18.1
```

3. 啟動 Express 伺服器

```
nodemon app.js
```

4. 安裝 express-handlebars

```
npm I express-handlebars@6.0.6
```

5. 設定在 Express 中使用的 express-handlebars
